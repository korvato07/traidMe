Source de Données
Récupération en temps quasi-réel

Yahoo Finance via yfinance

Toutes les données OHLCV (Open, High, Low, Close, Volume) proviennent de Yahoo Finance, une source gratuite et fiable pour les marchés financiers mondiaux. Les tickers utilisés :

GOLD → GC=F (Gold Futures CMX — corrélé 1:1 au spot XAU/USD) BTC → BTC-USD (Bitcoin / Dollar US) ICP → ICP-USD (Internet Computer Protocol) XRP → XRP-USD (Ripple XRP) BNB → BNB-USD (BNB Chain / Binance Coin)

Les données sont rafraîchies via WebSocket toutes les 30 secondes (badge ● LIVE) avec fallback polling 60s si la connexion WebSocket échoue. Un cache serveur de 55s évite de surcharger Yahoo Finance sur le dashboard multi-actifs.

Les 8 Indicateurs Techniques
Implémentés en pur pandas/numpy — aucune bibliothèque externe

1. RSI — Relative Strength Index (14 périodes)

Le RSI mesure la vitesse et l'amplitude des mouvements de prix pour identifier les zones de surachat et de survente.

Gain moyen = EWM(gains positifs, com=13) Perte moyenne = EWM(pertes positives, com=13) RS = Gain moyen / Perte moyenne RSI = 100 − (100 / (1 + RS))
  • RSI < 30 → Zone de survente : pression vendeuse excessive, rebond probable
  • RSI > 70 → Zone de surachat : pression acheteuse excessive, correction probable
  • 30 < RSI < 70 → Zone neutre, pas de signal fort

2. MACD — Moving Average Convergence Divergence (12, 26, 9)

Le MACD détecte les changements de momentum et les croisements de tendance.

EMA rapide = EMA(close, 12 périodes) EMA lente = EMA(close, 26 périodes) Ligne MACD = EMA rapide − EMA lente Ligne Signal = EMA(MACD, 9 périodes) Histogramme = MACD − Signal
  • MACD > Signal + histogramme positif → Élan acheteur, tendance haussière
  • MACD < Signal + histogramme négatif → Élan vendeur, tendance baissière
  • Plus l'histogramme est grand, plus l'élan est fort

3. EMA — Exponential Moving Averages (9, 21, 50)

Les EMA donnent plus de poids aux prix récents pour détecter la direction de la tendance.

EMA(n) = Prix × k + EMA_précédente × (1 − k) où k = 2 / (n + 1)
  • EMA9 > EMA21 > EMA50 → Alignement haussier : tendance montante confirmée sur les 3 horizons
  • EMA9 < EMA21 < EMA50 → Alignement baissier : tendance descendante confirmée
  • Configuration mixte → tendance indécise, signal faible
  • Le prix au-dessus de l'EMA50 indique un contexte globalement haussier

4. Bandes de Bollinger (20 périodes, 2 écarts-types)

Les bandes de Bollinger mesurent la volatilité et identifient les niveaux de support/résistance dynamiques.

Bande Médiane = SMA(close, 20) Écart-type = STD(close, 20) Bande Haute = Médiane + (2 × Écart-type) Bande Basse = Médiane − (2 × Écart-type)
  • Prix ≤ Bande basse → Rebond potentiel : le prix est statistiquement bas
  • Prix ≥ Bande haute → Résistance/retournement : le prix est statistiquement haut
  • Bandes serrées = faible volatilité (breakout imminent)
  • Bandes larges = forte volatilité en cours

5. ATR — Average True Range (14 périodes)

L'ATR mesure la volatilité absolue du marché, indépendamment de la direction. C'est la base du calculateur de risque.

True Range = max(High−Low, |High−Close_prev|, |Low−Close_prev|) ATR(14) = EWM(True Range, com=13)
  • ATR élevé → marché volatile → stop loss plus large nécessaire
  • ATR faible → marché calme → stop loss plus serré possible
  • Stop Loss suggéré = Prix actuel ± (2 × ATR)
  • Take Profit suggéré = Prix actuel ± (3 × ATR) → ratio risque/rendement 1:1.5

6. VWAP — Volume Weighted Average Price

Le VWAP est le prix moyen pondéré par le volume de la session. Référence clé des traders institutionnels.

Prix Typique = (High + Low + Close) / 3 VWAP = Σ(Prix Typique × Volume) / Σ(Volume)
  • Prix > VWAP → les acheteurs dominent la session, contexte haussier
  • Prix < VWAP → les vendeurs dominent la session, contexte baissier
  • Utilisé comme benchmark par les banques et fonds d'investissement
  • Visible sur le graphique via le bouton VWAP (ligne rose)

7. Ichimoku Cloud (9, 26, 52)

Indicateur japonais tout-en-un : tendance, momentum, support et résistance en un seul graphique.

Tenkan-sen = (Max(9) + Min(9)) / 2 ← tendance court terme Kijun-sen = (Max(26) + Min(26)) / 2 ← tendance moyen terme Senkou A = (Tenkan + Kijun) / 2 [décalé +26] Senkou B = (Max(52) + Min(52)) / 2 [décalé +26] Chikou = Close [décalé −26]
  • Prix au-dessus du nuage (SpanA & B) → tendance haussière forte
  • Prix en-dessous du nuage → tendance baissière forte
  • Prix dans le nuage → zone d'indécision, éviter d'entrer
  • Tenkan > Kijun → impulsion haussière (+0.5 score)
  • Visible via le bouton Ichimoku (4 lignes colorées)

8. Niveaux Pivot (Support & Résistance)

Niveaux de prix mathématiquement calculés à partir de la bougie précédente. Utilisés par des millions de traders, ils deviennent auto-réalisateurs.

Pivot = (High_prev + Low_prev + Close_prev) / 3 R1 = 2 × Pivot − Low_prev R2 = Pivot + (High_prev − Low_prev) R3 = High_prev + 2 × (Pivot − Low_prev) S1 = 2 × Pivot − High_prev S2 = Pivot − (High_prev − Low_prev) S3 = Low_prev − 2 × (High_prev − Pivot)
  • R1/R2/R3 → zones de résistance potentielle (prix plafonds)
  • S1/S2/S3 → zones de support potentiel (prix planchers)
  • Cassure de R2 avec volume → continuation haussière forte
  • Rebond sur S1 → opportunité d'achat en tendance haussière
  • Visible via le bouton Pivots (lignes horizontales en pointillés)

Moteur de Signal — Système de Score Pondéré
Chaque indicateur contribue au score final

Chaque indicateur génère un score positif (signal haussier) ou négatif (signal baissier). Les scores s'additionnent pour produire un score global :

ConditionScoreInterprétation
RSI < 25+3.0Survente extrême — signal fort
RSI < 30+2.0Survente — signal haussier
RSI > 75−3.0Surachat extrême — signal fort
RSI > 70−2.0Surachat — signal baissier
MACD haussier (fort)+2.0Élan acheteur
MACD baissier (fort)−2.0Élan vendeur
EMA9 > EMA21 > EMA50+1.5Tendance haussière alignée
EMA9 < EMA21 < EMA50−1.5Tendance baissière alignée
Prix touche bande Bollinger basse+1.5Support dynamique
Prix touche bande Bollinger haute−1.5Résistance dynamique
Tenkan > Kijun (Ichimoku)+0.5Impulsion haussière court terme
Prix au-dessus du nuage Ichimoku+0.8Tendance haussière confirmée
Tenkan < Kijun (Ichimoku)−0.5Impulsion baissière court terme
Prix en-dessous du nuage Ichimoku−0.8Tendance baissière confirmée

Score maximum théorique : ±8.6 (tous indicateurs alignés dans la même direction, y compris Ichimoku)

Règle de décision finale

Score ≥ +3.0 → ▲ ACHAT (BUY) Score ≤ −3.0 → ▼ VENTE (SELL) Sinon → ◆ NEUTRE (NEUTRAL) Confiance = 50% + (|score| × 8%) [plafonné à 100%]

Un score de ±7 (maximum théorique) donne une confiance de 100%. Un score de ±3 donne environ 74% de confiance.

Prévisions Multi-Horizons
Le même moteur appliqué sur 3 intervalles de temps différents

Pour chaque horizon, l'algorithme télécharge un jeu de données avec un intervalle adapté, calcule les indicateurs, et génère un signal indépendant :

Horizon 30 min
Bougies 5m
Données des 5 derniers jours
Horizon 1 heure
Bougies 15m
Données des 5 derniers jours
Horizon 2 heures
Bougies 30m
Données des 60 derniers jours

Les signaux peuvent diverger entre horizons — c'est normal et informatif. Ex : SELL sur 30 min mais BUY sur 2h indique une correction à court terme dans une tendance haussière plus large.

Limites & Précautions
Ce que l'algorithme ne fait pas
  • Analyse technique uniquement — Aucune analyse fondamentale (news, bilans, événements macro) n'est prise en compte.
  • Données légèrement différées — Yahoo Finance peut introduire un délai de 15 min sur certains marchés.
  • Backtesting simplifié disponible — L'endpoint /api/backtest?asset=BTC&interval=1h&period=60d applique le moteur sur les données historiques et calcule le taux de réussite. Résultat indicatif seulement — les performances passées ne garantissent pas les résultats futurs.
  • Marchés fermés — En dehors des heures de trading (or, futures), les données peuvent être obsolètes.
  • Signal ≠ Certitude — Un score de 100% de confiance signifie seulement que tous les indicateurs convergent, pas que le marché suivra.
